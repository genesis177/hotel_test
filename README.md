# AirBNB Backend

Надежное backend-приложение, имитирующее основные функции платформы типа AirBNB.  
Проект создан на Java и Spring Boot и включает управление отелями, бронирование номеров, динамическое ценообразование и обработку платежей.

---

# Основной функционал

## Управление отелями и комнатами
- Создание и управление несколькими отелями
- Поддержка различных типов комнат:
  - Standard
  - Deluxe
  - Suite
- Настройка вместимости и базовой стоимости комнат
- Возможность активировать и деактивировать отели или комнаты

---

## Система инвентаря и цен
- Отслеживание доступности комнат по датам
- Динамическое изменение цен через `surgeFactor`
  - например повышение цен в выходные или праздники
- Защита от двойного бронирования
- Проверка доступности номеров в реальном времени

---

# Система бронирования
- Создание и управление бронированиями
- Поддержка статусов:
  - CONFIRMED
  - CANCELLED
  - COMPLETED
- Привязка нескольких гостей к одному бронированию
- Расчет стоимости проживания по датам заезда и выезда

---

# Пользователи и платежи
- Привязка платежей к бронированиям
- Поддержка ролей:
  - Guest
  - Host
  - Admin
- Хранение информации о транзакциях

---

# Архитектура проекта

Проект построен на реляционной базе данных с акцентом на:
- масштабируемость
- надежность
- целостность данных

## Основные сущности

### Hotel
Информация об отеле.

Связи:
- один отель → много комнат
- один отель → много записей inventory
- владелец отеля → User

### Room
Тип комнаты внутри отеля.

### Inventory
Контролирует:
- доступность комнат
- цену на конкретную дату

### Booking
Информация о бронировании пользователя.

### User
Пользователи системы:
- гости
- владельцы
- администраторы

### Payment
Данные о платежах и транзакциях.
